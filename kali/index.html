Introduction

One of the first CSE groups I found at Michigan State was a little group called "Defense against the Dark Arts", a grad student really enjoyed his work with computer security and hosted a couple meetings where he demonstrated a few hacking examples. When I went, everyone was dual-booting their laptops with something called BackTrack (later I learned that was a Linux OS that is the prelude to Kali). This is where I learned about Offensive Security's new system Kali.

Now, Kali is a Linux system based on Debian. It's used for digital forensics and penetration testing[1]. Basically, its a collection of some of the security worlds finest tools of the trade and we can use Kali for all these tools in one central place. One machine can be used to map a network, find the router, and start intercepting passwords and data for instance.

Installation

If you're trying to get your hands on a distribution of Kali, it's really simple (or so I'm told). The basic idea is that you download an image (.iso file) and that's the blueprints to the new operating system.

You have a few options based on the hardware or computer experience that you have. For instance you could save the system to a usb flash drive and boot from that when you start any machine. You could dual-boot your laptop. Or you could have virtual machines inside of a OS you already use (ie windows). There are so many options that it's a really good idea to research what would be best before you try to install something and possibly break the only thing you can use to write next week's exam.
